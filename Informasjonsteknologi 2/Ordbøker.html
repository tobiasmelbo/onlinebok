Ordbøker#

Ordbøker (dictionaries) er en måte å lagre data. I Lister lagrer vi data i en rekkefølge. I ordbøker lagrer vi data i par med keys og values, nøkler og verdier. En nøkkel er forbundet med sin verdi.

land = {"NO" : "Norge", "SE" : "Sverige"}

print(land["NO"])
print(land["SE"])

Norge
Sverige

I koden over lager vi landkoder. "NO" og "SE" er nøklene, og "Norge" og "Sverige" er de tilhørende verdiene. Vi knytter sammen nøklene og verdiene med kolon :. For å hente verdier fra ordboken land så skriver vi inn nøkkelen med land["NO"] og så får vi ut verdien "Norge".

Dersom vi skal sette sammen en enkel ordbok så er det vanlig å gjøre det over flere linjer. Det er litt mer oversiktlig enn eksemplet over.

land = {
    "NO" : "Norge",
    "SE" : "Sverige"
}

Nøklene og verdiene i en ordbok trenger ikke være strings!

Nøklene og verdiene i en ordbok kan være av mange forskjellige typer.

mai = {
    17 : "Nasjonaldag!",        # Nøkkelen 17 har verdien "Nasjonaldag!"
    "erVår" : True,             # Nøkkelen "erVår" har verdien True
    True : "Det er mai!"        # Nøkkelen True har verdien "Det er mai!"
}

print(mai[17])
print(mai["erVår"])
print(mai[True])

gir

Nasjonaldag!
True
Det er mai!

Legge til, endre og fjerne data#

For å legge til data så setter vi en ny nøkkel lik en verdi.

land = {}                # Tom ordbok

land["NO"] = "Norge"     # Verdien "Norge" legges inn i ordboken under nøkkelen "NO"
land["DK"] = "Danmark"   # Verdien "Danmark" legges inn i ordboken under nøkkelen "DK"

print(land)

{'NO': 'Norge', 'DK': 'Danmark'}

For å endre på data kan vi sette en eksisterende nøkkel til en ny verdi.

land = {
    "NO" : "Norge",
    "DK" : "Danmark"
} 

print(land)

land["DK"] = "Donkey Kong" # Nøkkelen "DK" får en ny verdi "Donkey Kong" 🐒

print(land)

{'NO': 'Norge', 'DK': 'Danmark'}
{'NO': 'Norge', 'DK': 'Donkey Kong'}

Vi kan fjerne data ved å bruke pop()-metoden.

land = {
    "NO" : "Norge",
    "SE" : "Sverige",
    "DK" : "Danmark"
}

land.pop("NO") # Fjerner både nøkkel og verdi for nøkkelen "NO"

print(land)

{'SE': 'Sverige', 'DK': 'Danmark'}

Løkker og ordbøker#

Vi kan gå igjennom ordbøker med for-løkker på denne måten.

land = {
    "NO" : "Norge",
    "SE" : "Sverige"
}

for x in land:
    print("Nøkkel:", x)         # Skriver ut nøkkelen
    print("Verdi:", land[x])   # Skriver ut verdien til nøkkelen

Nøkkel: NO
Verdi: Norge
Nøkkel: SE
Verdi: Sverige

Lister av ordbøker#

Det kan være nyttig å lage flere ordbok-objekter og lagre de i en liste.

La oss lage en liste elevliste som skal inneholde ordbøker med fag som nøkler og tilfeldige karakterer som verdier.

from random import randint

elevliste = []

for n in range(3):
    # Lager ordboken elev
    elev = {}

    # Legger inn karakterer
    elev["Norsk"] =  randint(1, 6)
    elev["Gym"] = randint(1, 6)
    elev["Matte"] = randint(1, 6)

    # Legger ordboken inn i listen
    elevliste.append(elev)

print(elevliste)

[{'Norsk': 3, 'Gym': 2, 'Matte': 1}, {'Norsk': 3, 'Gym': 6, 'Matte': 2}, {'Norsk': 5, 'Gym': 1, 'Matte': 6}]

Som vi ser har vi nå laget en liste som er fylt med ordbøker. Hvert element i listen skal representere en tilfeldig elev.

Oppgaver#

Oppgave 1#

Lag en ordbok elev som skal ha fag som nøkler og karakterer som verdier. Den kan starte slik

elev = {
    "Norsk" : 4,
    "Gym" : 3
}

  1. Legg til fire nye fag og karakterer. Du kan bestemme både fag og karakterer.

  2. Lag en løkke som skriver ut alle fagene og karakterene med en setning Eleven har <karakter> i faget <fag>.

  3. Finn og skriv ut karaktersnittet til eleven.

Oppgave 2#

Ordbøker kan også ha lister og andre ordbøker som verdier.

klasse = {
    "Navn" : "10F",
    "Fag" : "Naturfag",
    "Karakterer" : [4, 5, 2, 5, 2, 1, 6, 3, 4, 3]
}

Finn gjennomsnittet av karakterene ved å bruke ordboken klasse.

Oppgave 4#

En vanlig måte å lagre farger på er ved å oppgi en verdi for hver farge. Tallene er heltall i intervallet \([0,255]\). Vi bruker "R" for rød, "G" for grønn og "B" for blå.

Lag en liste farger som skal inneholde \(5\) tilfeldig genererte farger som ordbøker.

Nøklene til ordbøkene skal være "R", "G" og "B". Disse skal ha tilfeldige verdier i intervallet \([0,255]\).

En del av utskriften til listen farger vil kunne se ut som dette: [{'R': 124, 'G': 52, 'B': 230}, {'R': 32, 'G': 89, 'B': 10}, ...]
