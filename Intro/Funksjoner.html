Funksjoner#

Vi kan tenke på en funksjon som en oppgave som kan inneholde flere instruksjoner. For å kjøre en funksjon gjør vi et kall på funksjonen.

def hils():
    print("Hei!") # Skriver ut "Hei!"

hils()            # Et kall på funksjonen hils().

Hei!

Når vi skriver hils() gjør vi et kall på funksjonen. Da gjennomføres det som står med innrykk.

Parametre og return-verdier#

Ofte ønsker vi å sette variabler inn i en funksjon, la gjøre funksjonen gjøre noe, og så få ut noe.

Det vi setter inn kalles funksjonens parameter. Det vi får ut kalles funksjonens return-verdi.

def dobbelt(x):   # x er navnet på funksjonens parameter
    return 2*x    # return-verdien er 2 ganger x.

print(dobbelt(2)) # 2 settes inn som parameter til funksjonen

4

Når vi kjører funksjonen med funksjonskallet dobbelt(2) så skjer dette:

  • 2 settes inn i funksjonen som x

  • funksjonen returnerer to ganger x, som blir 4

Derfor skrives 4 ut.

Flere parametre#

Vi kan også gi en funksjon flere parametre ved å skille de med komma.

def størst(a, b):
    if a >= b:
        return a
    else:
        return b
    
print(størst(2, 3))
print(størst(-2, 1))

3
1

Denne funksjonen returnerer det som er størst av enten a eller b.

Oppgave 2#

Med Pytagoras læresetning kan vi finne den lengste siden i en rettvinklet trekant med formelen \(c = \sqrt{a^2+b^2}\).

Lag en funksjon som tar lengden av to sider a og b som parameter og returnerer lengden av den lengste siden c.

Test funksjonen med trekanten a = 3 og b = 4. Dette skal gi 5.

Oppgave 3 (utfordring)#

Pytagoreiske tripler er tre og tre tall som passer i likningen \(a^2 + b^2 = c^2\). Det enkleste eksemplet er \(a=3\), \(b=4\) og \(c=5\). Vi kan skrive det som \((3,4,5)\).

a) Lag en funksjon som sjekker om tre heltall a, b og c er pytagoreiske tripler. Funksjonen skal returnere True hvis det er en pytagoreisk trippel, og False hvis ikke.

b) Skriv ut alle pytagoreiske tripler bestående av heltall under \(100\). Bruk funksjonen som hjelp til dette.

Primitive pytagoreiske tripler er de triplene som ikke deler en felles faktor større enn \(1\). Med andre ord er \((3,4,5)\) en primitiv pytagoreisk trippel, men ikke \((6,8,10)\) fordi \(6\), \(8\) og \(10\) deler en faktor \(2\).

c) Modifiser programmet til å bare skrive ut de primitive pytagoreiske triplene.
